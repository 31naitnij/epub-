# AI EPUB 电子书翻译助手

这是一个简单易用的 **EPUB 电子书翻译工具**。它的目标是：让你在翻译电子书内容的同时，**100% 保留原书的排版、插图和样式**。

---

## ✨ 为什么选择它？

- **排版零损坏**：只翻译文字，书里的图片、字体、表格、排版布局完全不变。
- **翻译更聪明**：专门优化的标识符技术，让 AI 能够精准区分正文和内部代码，翻译更准确。
- **简单直观**：三栏设计，左边选章节，中间看原文，右边实时对比检查。
- **断点续傳**：翻译一半网络断了？没关系，重新启动软件，之前的进度都在。

---

## 🚀 怎么使用？

1. **启动软件**：在文件夹里双击 `run.bat`。
2. **选择书籍**：点击“选择文件”，找到你想翻译的 `.epub` 电子书。
3. **划分段落**：点击 **“分块并分组”**，软件会自动把书拆成适合 AI 阅读的小块。
4. **填入 API 信息**：填写你的 AI 密钥（Key）和地址。
5. **开始翻译**：点击 **“开始翻译”**。你会看到右侧进度不断增加。
6. **导出书籍**：翻译完成后，点击 **“导出”**。一本翻译好的新书就会出现在你的文件夹里！

---

## 💡 小提示

- **导出失败怎么办？**：如果在点击“导出”时报错，通常是因为你正在用阅读器（如微信读书查看器、Calibre、Apple Books）打开那本生成的文件。请**关闭阅读器**后再试。
- **翻译封面？**：为了保护电子书的封面和技术文件不被打乱，软件会自动跳过这些特殊页面。
- **书里有数学公式？**：别担心，软件会自动保护公式代码，AI 只会翻译剩下的文字。

---

## 🔍 关于翻译中出现的特殊符号

在翻译过程中，你会在原文和译文框里看到一些奇怪的符号（如 `⟬⟭`, `⦗1⦘`, `⟦ ⟧` 等），请**不要删除或修改它们**：

- **它们是什么？**：这些是给 AI 看的“定位锚点”。它们代表了书中的加粗、倾斜、链接、换行或图片等格式信息。
- **为什么要用它？**：有了这些符号，AI 就能知道哪里需要保留原有的格式，从而保证翻译后的排版和原书一模一样。
- **导出会出现在书里吗？**：**绝对不会**。在最后点击“导出”时，软件会自动把这些符号变回原来的精美排版，你在手机或 Kindel 上看书时是看不到这些符号的。

## 📝 提示词示例 (Prompt Example)

为了获得最佳的翻译效果，建议在软件的“提示词”框中输入类似以下的内容：

> 你是一位专业的图书翻译，请将以下电子书内容翻译成中文。
> 
> **重要规则：**
> 1. 请保持文本中的所有特殊符号（如 `⟬⟭`, `⦗1⦘`, `⟦ ⟧` 等）**原封不动**地保留在译文的对应位置。
> 2. 只翻译符号之间的文字，不要尝试翻译或修改这些符号。
> 3. 译文要自然流畅，符合中文阅读习惯。
