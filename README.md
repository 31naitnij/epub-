# 📖 AI 翻译神器 - EPUB/DOCX 全能版

> 这是新手借助 Google Antigravity 开发的文档翻译工具。



## ✨ 它能干什么？

1.  **格式完美保留**：
    *   **EPUB 电子书**：翻译完还是 EPUB，目录、排版、甚至原来的图片css样式都不变。
    *   **DOCX 文档**：把英文文档变成中文文档，保持段落结构。
    *   **Output 自由选**：你可以把 EPUB 翻译成 DOCX 打印出来，也可以把 DOCX 翻译成 EPUB 在手机上看！

2.  **超聪明的“上下文记忆”**：
    *   它不是一句一句死板翻译，而是会“记得”这页书前面说了什么（可以设置记 1-5 轮）。
    *   这样翻译出来的小说，人名、称呼、语气都连贯，不像机翻那样前言不搭后语。

3.  **随时“暂停/继续”**：
    *   几百万字的大部头翻译到一半电脑没电了？没关系！
    *   下次打开，它自动知道翻译到哪了，直接接着翻，不浪费一分钱 API 额度。

4.  **支持超多格式**：
    *   **输入支持**：EPUB, DOCX, Markdown, PDF (文字版), TXT, ODT
    *   **输出支持**：EPUB, DOCX (A4打印版), Markdown

## 📦 准备工作（只要这一步！）

为了让它能处理 DOCX、PDF 等高级格式，你需要安装一个超厉害的小工具叫 **Pandoc**。

1.  **下载 Pandoc**: [点击这里去官网下载](https://pandoc.org/installing.html) (找 Windows 的 `.msi` 安装包)。
2.  **安装它**: 双击安装，**记得一路“Next”，不需要改设置**。
3.  **搞定！** 就这么简单。(如果你只翻译 EPUB 并且只导出 EPUB，这一步甚至可以跳过，但强烈建议装上以获得完整体验)。

## 📥 下载与运行

如果你是从 Release 下载的 **.exe** 文件：

1.  **下载**：在页面右边的 **Releases** 里下载最新的压缩包。
2.  **解压**：把下载的包解压到一个文件夹里。
3.  **运行**：双击里面的 `AI_EPUB_Translator.exe`。
    *   *不需要安装 Python，也不需要配环境，点开即用！*
4.  **注意**：如果是 Windows 10/11，可能会弹出安全警告，点击“更多信息”->“仍要运行”即可。

## 🚀 怎么用？（超简单）

### 第一步：设置好
1.  打开软件。
2.  **选文件**：点第一个按钮选你要翻译的书 (支持 .epub, .docx, .md, .pdf, .txt)。
3.  **填 Key**：在右边填上你的 AI API Key 和 URL (比如 OpenAI, DeepSeek 等)。
    *   *不知道这是啥？去问问卖给你 API 的服务商~*

### 第二步：分好块
1.  点一下底部的 **“仅分块处理”**。
2.  你会看到左边的表格里列出了书里的每一段内容。
3.  **小技巧**：你可以随便点一行，右边就会显示这段的原文。你可以在翻译前就看到书的内容。

### 第三步：开始翻！
你有两个选择：
*   **我想先试试**：按住 Ctrl 或 Shift 键，在列表里随便选几行，然后点 **“翻译选中块”**。看看效果满不满意。
*   **直接梭哈**：直接点 **“全部翻译”**，然后去喝杯咖啡吧。

### 第四步：导出
等进度条跑完，或者你觉得翻译得差不多了：
1.  在上面选一个 **“输出格式”** (比如你想把 EPUB 转成 Word 就选 DOCX)。
2.  点 **“导出”**。
3.  去输出目录拿你的热乎乎的译文吧！

## ❓ 常见问题

**Q: 那个“上下文轮数”是干嘛的？**
A: 设为 1，它只看前一段；设为 5，它能记得前 5 段的情节。一般小说建议设 2-3，太高了会费点钱。

**Q: 我翻译了一半关掉了，下次怎么继续？**
A: 只要你不点“清除缓存”，下次打开同一个文件，它会自动检测到“诶？这本书翻译过！”，你直接点“全部翻译”，它会瞬间跳过已经翻译好的部分，从没翻译的地方开始。仅需几秒钟！

**Q: 输出格式里的 Auto 和 EPUB 有啥区别？**
A: 如果你是 EPUB 转 EPUB，选谁都一样。如果你是 DOCX 想转 EPUB，那就选 EPUB。Auto 就是“原来啥样出来就啥样”。
