# 📖 AI 文档翻译工具 - EPUB/DOCX

> 这是新手借助 Google Antigravity 开发的文档翻译工具。
支持epub、docx



## 📦 请先安装 Pandoc

你需要安装 **Pandoc**。

1.  **下载 Pandoc**: [点击这里去官网下载](https://pandoc.org/installing.html) (找 Windows 的 `.msi` 安装包)。
2.  **安装它**: 双击安装，**记得一路“Next”，不需要改设置**。
3.  **搞定！** 就这么简单。(如果你只翻译 EPUB 并且只导出 EPUB，这一步甚至可以跳过，但强烈建议装上以获得完整体验)。

## 📥 下载与运行

如果你是从 Release 下载的 **.exe** 文件：

1.  **下载**：在页面右边的 **Releases** 里下载最新的压缩包。
2.  **解压**：把下载的包解压到一个文件夹里。
3.  **运行**：双击里面的 `AI_EPUB_Translator.exe`。
    *   *不需要安装 Python，也不需要配环境，点开即用！*
4.  **注意**：如果是 Windows 10/11，可能会弹出安全警告，点击“更多信息”->“仍要运行”即可。

## 🚀 使用说明

### 第一步：设置好
1.  打开软件。
2.  **选文件**：点第一个按钮选你要翻译的书 (支持 .epub, .docx, .md, .pdf, .txt)。
3.  **填 Key**：在右边填上你的 AI 模型名、API Key 和 URL (比如 OpenAI, DeepSeek 等)。
    *   需要仔细阅读你的ai api文档获取准确信息。


### 第二步：分块
1.  填写分块字符数，建议2000即可，根据模型能力调整。
2.  点一下底部的 **“仅分块处理”**。
3.  你会看到左边的表格里列出了书里的每一段内容。
4.  **小技巧**：你可以随便点一行，右边就会显示这段的原文。你可以在翻译前就看到书的内容。

### 第三步：开始翻译
两个选择：
*   **翻译所选**：按住 Ctrl 或 Shift 键，在列表里，然后点 **“翻译选中块”**。
*   **翻译全部**：直接点 **“全部翻译”**，然后等待。

### 第四步：导出

1.  在上面选一个 **“输出格式”** (比如你想把 EPUB 转成 Word 就选 DOCX)。
2.  点 **“导出”**。
3.  去输出目录拿你的热乎乎的译文吧！

## ❓ 常见问题

**Q: “上下文轮数”如何设置？**
A: 设为 1，它只看前一段；设为 5，它能记得前 5 段的情节。一般1就可以，越严谨就越高，太高费钱。

**Q: 断点续传**
A: 只要你不点“清除缓存”，下次打开同一个文件，它会自动检测到“诶？这本书翻译过！”，你直接点“全部翻译”，它会瞬间跳过已经翻译好的部分，从没翻译的地方开始。仅需几秒钟！

**Q: 输出格式里的 Auto 和 EPUB 有啥区别？**
A: 如果你是 EPUB 转 EPUB，选谁都一样。如果你是 DOCX 想转 EPUB，那就选 EPUB。Auto 就是“原来啥样出来就啥样”。
